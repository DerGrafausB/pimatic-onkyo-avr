This plugin lets you control your Onkyo and Pioneer (model year 2016+) AVRs

# Configuration

### Sample Device Config:
```javascript
    {
      "id": "onkyo-avr",
      "name": "Onkyo AVR",
      "class": "OnkyoAVRDevice",
      "ip": "192.168.0.15"
    },
```
